# 🎓 All-in-One Result Management (Bangladesh)

A complete result management system for schools & colleges — classes, sections, customizable
subjects & grading (BD GPA rules), CSV/Excel upload, merit lists and printable A4 marksheets.

## ▶ Open the app

**https://humayun2000444.github.io/allinoneresult-app/**

Everything runs in your browser — your data stays on **your own computer** and is never uploaded anywhere.

## 💾 Offline use (recommended for schools)

Download **[AllInOneResult.html](./AllInOneResult.html)** (right-click → Save link as…),
copy it to any Windows PC and just **double-click** it. No internet or installation needed.

Demo files to try the upload flow: [demo-result.csv](./demo-result.csv) (marks) ·
[demo-students.csv](./demo-students.csv) (student list).
